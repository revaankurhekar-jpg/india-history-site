# 🇮🇳 History of India Website

## 📌 Project Overview

This is a simple and interactive website that explains the history of India in a structured and easy-to-understand way.
The website allows users to explore different historical periods by clicking buttons, making it more engaging than a normal static page.

---

## ✨ Features

* 🟢 Welcome screen on first load
* 🟠 Click buttons to view different historical periods
* 🔵 Sections include:

  * Ancient India
  * Medieval India
  * British Rule
  * Independence Movement
* 🔁 Back button to return to home screen
* 🎨 Clean and modern design
* ⚡ Works offline (no internet required)

---
